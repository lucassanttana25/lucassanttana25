## 🚀 Projeto em Destaque: **Guardia**
O **Guardia** é uma plataforma robusta e moderna para gestão de carteira de investimentos e ativos financeiros. 

- **🔗 Dashboard Online:** [Acesse o Guardia Dashboard](http://137.131.251.127/dashboard)
- **Arquitetura & Stack:**
  - **Backend:** NestJS, TypeScript, Prisma ORM, PostgreSQL.
  - **Frontend:** React, Vite, TypeScript, Tailwind CSS, TanStack Router/Query, Shadcn UI.
  - **Serviços & Integrações:** Autenticação via Clerk, Cache de alta performance com Redis, Envio de e-mails via Resend.
  - **Recursos Principais:**
    - Autenticação de usuários e rotas 100% protegidas.
    - CRUD de carteiras e ativos com cálculo automático de **Preço Médio**.
    - Sistema de cotações em tempo real com fallback multi-provedor (Yahoo Finance / Brapi) e cache resiliente.
    - Motor de alertas por preço-alvo e geração de relatórios semanais consolidados agendados via cron job.
